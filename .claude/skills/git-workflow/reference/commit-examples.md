# Commit Examples by Stack

## React / TypeScript

```
feat(auth): add login form with email validation
feat(dashboard): create KPI summary cards component
fix(router): handle expired token redirect loop
fix(form): prevent double submit on slow network
refactor(hooks): extract useDebounce from search component
test(auth): add login flow integration tests
style(components): align button sizes with DS tokens
chore(deps): upgrade react-router to v7
perf(table): virtualize rows for large datasets
```

## .NET / C#

```
feat(api): add user registration endpoint
feat(cqrs): implement CreateOrderCommand handler
fix(repo): handle concurrent update conflict in UserRepository
fix(auth): validate JWT expiry before token refresh
refactor(services): extract IEmailService interface
test(integration): add API endpoint tests with WebApplicationFactory
chore(deps): update Microsoft.EntityFrameworkCore to 9.0
perf(query): add composite index for order lookups
```

## Python

```
feat(api): add /users endpoint with FastAPI router
feat(agent): create document summarization LangGraph flow
fix(db): handle connection pool exhaustion on high load
fix(auth): sanitize JWT claims before storage
refactor(services): extract base repository pattern
test(api): add pytest fixtures for user endpoints
chore(deps): pin langchain to 0.3.x
perf(cache): add Redis caching for user profile lookups
```

## Flutter / Dart
